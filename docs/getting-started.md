# exists.lol

## Get started

1. Fork the repository
2. Create `domains/yourname.json`
3. Fill the template and choose the type (Supported types [here](/records.md)):
#### Template:
```json
{
  "owner": {
    "username": "<your_github_username>",
    "email": "<your_email>"
  },
  "records": {
    "<record_type>": "<your_target>"
  }
}
```
4. Save everything
5. Create a pull request [here](https://github.com/segfaultuwu/exists.lol/pulls)
6. Wait. our maintainers will try their best to check and merge your pull request as fast as possible

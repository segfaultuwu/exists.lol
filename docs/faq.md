# FAQ

Frequently asked questions about `exists.lol`.

## What is exists.lol?

exists.lol is a free subdomain registry.

It lets you get a subdomain like:

`yourname.exists.lol`

and point it to your website, portfolio, bot, API, server, or project.

## Is it free?

Yes. Subdomains are free.

You only need a GitHub account to open a pull request.

## How do I get a subdomain?

Fork the repository, add a JSON file inside the `domains` folder, and open a pull request.

After your pull request is reviewed and merged, your subdomain will go live.

## Where do I put my domain file?

Put it inside the `domains` folder.

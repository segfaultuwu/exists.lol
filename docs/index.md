# exists.lol

Grab your own clean, sweet-looking `*.exists.lol` subdomain.

exists.lol is a free subdomain registry for websites, portfolios, projects, bots, APIs, and random cool stuff.

## Example

`yourname.exists.lol`

## How it works

1. Fork the repository.
2. Add your domain file inside the `domains` folder.
3. Open a pull request.
4. Wait for review.
5. After your PR is merged, your subdomain will go live.

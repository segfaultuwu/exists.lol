## What is this?

> exists.lol is a free subdomain registry inspired by [is-a.dev](https://github.com/is-a-dev/register).

You can get a subdomain like:

<p align="center">
  <code>yourname.exists.lol</code>
</p>

and point it to your GitHub Pages site, server, VPS, API, bot dashboard, portfolio, or project.

## How it works

1. Fork this repository.
2. Add your domain file inside the <code>domains</code> folder.
3. Open a pull request.
4. Wait for review.
5. After your PR is merged, your subdomain will go live.

---

## Rules

- No phishing, malware, scams, spam, or illegal content.
- No impersonation of companies, projects, or other people.
- No reserved names like <code>www</code>, <code>api</code>, <code>mail</code>, <code>admin</code>, <code>root</code>, or <code>support</code>.
- No wildcard records.
- Do not mix <code>CNAME</code> with other record types.
- Keep it clean and useful.

---

## Supported records

exists.lol supports common DNS records:

<p>
  <code>A</code>
  <code>AAAA</code>
  <code>CNAME</code>
  <code>TXT</code>
  <code>MX</code>
</p>

For simple websites hosted on GitHub Pages, use a <code>CNAME</code> record.
